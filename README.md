# HR Analysis Dashboard

#  Overview

This project is an **HR Analytics Dashboard** built to analyze employee data and provide insights into **workforce distribution, performance, satisfaction, promotions, and attrition**.
It helps organizations make data-driven HR decisions by visualizing key trends across departments and employee attributes.



# 📊 Key Metrics (KPIs)

*  **Total Employees:** 11,428
*  **Average Working Hours / Month:** 201.05
*  **Promoted Employees (Last 5 Years):**
  * Promoted: **300 (2.63%)**
  * Not Promoted: **11,128 (97.37%)**

# Key Insights

* Promotions are **very limited (~2.6%)**, indicating potential retention or growth issues
* Majority of employees are concentrated in **Sales & Technical departments**
* Most employees handle **3–4 projects**, showing balanced workload distribution
* Salary distribution is heavily skewed toward **low & medium levels**
* Employee tenure peaks at **2–3 years**, suggesting moderate turnover
